## 🚀 筑基上手｜丹方实操指南
### 前置条件
- 已习得《仙门基础丹术》（掌握基础中药知识）
- 备齐炼丹炉（煎药壶）、灵草（药材）、控火符（燃气灶）

### 炼丹步骤（用药步骤）
1. 采集灵草（采购药材）：
   ```bash
   # 凡界操作：按药典采购道地药材
   采购 筑基培元丹原料：黄芪10g、白术10g、党参10g
   ```
2. 净化灵草（清洗药材）：
   - 用山泉水洗净灵草，去除杂质（凡界：清水洗净，去除泥沙）
3. 入炉炼丹（煎药）：
   - 丹炉加水3升，中火炼1炷香（凡界：加水500ml，小火煎30分钟）
4. 出丹服用（服药）：
   - 每日辰时服1粒，连服7日（凡界：每日早晚各1次，连服1周）

## ❓ 渡劫答疑｜常见用药误区
### Q1：炼筑基培元丹时，灵草越多越好？
A：非也！灵草过量易生丹毒（凡界：用药过量易致不良反应），按《仙门药典》配比即可，多炼无益。

### Q2：不同丹方可以一起炼？
A：忌！如筑基培元丹+清热解毒丹同炼，易致丹力相冲（凡界：多种中成药同服，易成分冲突），需间隔2炷香以上（2小时）。

### Q3：假灵草（假药）能炼出真丹？
A：绝无可能！假灵草炼出的丹，轻则无效，重则走火入魔（凡界：假药无药效，还可能伤身体），务必找正规仙门商铺采购（正规药店）。
